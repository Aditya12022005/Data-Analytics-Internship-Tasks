# Data Analytics Internship

This repository contains multiple data analytics and machine learning tasks completed as part of an internship program. Each task focuses on solving real-world problems using Python.

## Tasks Included
- Task 1: Exploratory Data Analysis (EDA)
- Task 2: Customer Segmentation
- Task 3: Data Cleaning
- Task 4: House Price Prediction
- Task 5: Wine Quality Prediction
- Task 6: Fraud Detection

Each task is organized in a separate folder with datasets, notebooks, and documentation.
