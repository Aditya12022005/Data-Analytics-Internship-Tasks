# Task 3(Level 1) : Data Cleaning Project 

## Project Description
Data cleaning is the process of fixing or removing incorrect, corrupted, duplicate, or incomplete data. This project demonstrates essential data cleaning techniques using real-world CSV and JSON datasets.

## Datasets Used
### Dataset 1: Airbnb NYC 2019
- File: `AB_NYC_2019.csv`
- Data includes listing prices, room types, locations, and availability.

### Dataset 2: Category JSON Dataset
- File: `CA_category_id.json`
- Contains nested category information in JSON format.

## Key Data Cleaning Steps
- Data integrity checks
- Handling missing values
- Removing duplicate records
- Standardizing categorical data
- Outlier detection and treatment
- Cleaning nested JSON data

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Output Files
- `AB_NYC_2019_cleaned.csv`
- `CA_category_cleaned.csv`
